## 📑 Descrição
Projeto que permite gerar QR Codes, senhas e criptografar via terminal.

## 🎯 Funcionalidades
- [X] : Gerar QR Codes.
- [X] : Gerar senhas.
- [X] : Criptografar senhas ou textos.

- `Node.js`
- `Javascript`

## 🛠️ Como abrir e rodar o projeto

1. Run `npm run start`
2. OU
